# Brief: listing images

Source: client proposal (Marlowe & Hart)

Admins currently add listing photos by pasting image URLs. Replace that with direct image uploads that Doorstead stores itself.

Requirements:
- Upload multiple image files per listing in one go.
- Reorder images; the first is the cover/hero shown on the card and listing page.
- Mark one image as the cover, and mark one as the floorplan.
- Keep existing pasted-URL listings working; don't break them.
- Enforce sensible file-type and size limits.
